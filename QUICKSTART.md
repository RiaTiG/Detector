# Быстрый старт

## Проблема с Python 3.14

У вас Python 3.14, который несовместим с библиотекой spaCy (используется Pydantic v1).

**Решение:** Проект автоматически использует облегченную версию `feature_extractor_lite.py`, которая работает без spaCy.

## Запуск примера

```bash
# 1. Установите зависимости (если еще не установлены)
pip install -r requirements.txt

# 2. Запустите пример
python example.py
```

Пример создаст:
- Тестовый датасет в `data/sample/`
- Обученную модель
- Визуализации в `results/`

## Что было исправлено

1. **Создан `feature_extractor_lite.py`** - работает без spaCy
2. **Обновлены импорты** - автоматически используют lite версию
3. **34 признака** вместо 25 (добавлены дополнительные стилометрические признаки)

## Основные отличия lite версии

### Работает ✅
- Все стилометрические признаки
- Базовые синтаксические признаки (POS-теги через NLTK)
- Семантические признаки (местоимения, модальные глаголы, и т.д.)

### Не работает (требует spaCy) ❌
- Глубина синтаксического дерева зависимостей
- Точная длина зависимостей
- Именованные сущности (NER)

## Если нужна полная версия со spaCy

Используйте Python 3.11 или 3.12:

```bash
# Создайте виртуальное окружение с Python 3.12
python3.12 -m venv venv
source venv/bin/activate  # Linux/Mac
# или
venv\Scripts\activate  # Windows

# Установите зависимости
pip install -r requirements.txt
python -m spacy download en_core_web_sm

# Удалите _lite импорты и используйте стандартный feature_extractor.py
```

## Признаки в lite версии

**Стилометрические (18 признаков):**
- Длина предложений (среднее, дисперсия, мин, макс)
- Type-Token Ratio
- Hapax/Dis legomena
- Стоп-слова
- Пунктуация (общая, запятые, точки с запятой и т.д.)
- Длина слов

**Синтаксические (9 признаков):**
- Запятые на предложение
- Скобки, кавычки
- POS-теги (существительные, глаголы, прилагательные, наречия)

**Семантические (7 признаков):**
- Местоимения (общие, первого лица)
- Отрицания
- Модальные глаголы
- Сокращения (can't, I'm и т.д.)
- Интенсификаторы (very, really)
- Дискурсивные маркеры (well, so, like)
